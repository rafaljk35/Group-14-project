Example code taken from this website: https://dronebotworkshop.com/esp32-servo/
I've added my own comments to it to hopefully make operation clearer

The library used is from here: https://github.com/madhephaestus/ESP32Servo
Use the link to view the documentation

To install the library in the Arduino IDE:
1. Go to Sketch > Include Library > Manage Libraries
2. Type ESP32Servo into search box
3. Find the library named "ESP32Servo" by Kevin Harrington, John K. Bennett
4. Click Install
